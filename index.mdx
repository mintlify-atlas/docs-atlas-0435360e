ESP-IoT-Solution is Espressif's official collection of IoT components that extend [ESP-IDF](https://github.com/espressif/esp-idf) with ready-to-use device drivers, communication protocols, and application frameworks. Whether you're building a smart home device, industrial controller, or wearable, ESP-IoT-Solution provides the building blocks you need.

## Get started

<Steps>
  <Step title="Set up ESP-IDF">
    Install the ESP-IDF development framework (v5.3 or later for the master branch). Follow the [ESP-IDF Getting Started Guide](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/get-started/index.html).
  </Step>
  <Step title="Add a component">
    Use the ESP Component Registry to add any component to your project:

    ```bash
    idf.py add-dependency "espressif/button"
    ```
  </Step>
  <Step title="Build and flash">
    Build your project and flash it to your ESP32 board:

    ```bash
    idf.py build
    idf.py -p /dev/ttyUSB0 flash monitor
    ```
  </Step>
</Steps>

<Note>
  All components in ESP-IoT-Solution are available on the [ESP Component Registry](https://components.espressif.com/). You can add them to any ESP-IDF project without cloning this repository.
</Note>
